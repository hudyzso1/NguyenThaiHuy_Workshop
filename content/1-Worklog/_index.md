This section summarizes my internship worklog during the **First Cloud AI Journey** program.

The internship lasted for 12 weeks, from **May 5, 2026 to July 27, 2026**. During this period, I learned about AWS Cloud services, practiced cloud deployment, worked on the CyberNet project, documented the implementation process, and improved the system based on feedback.

The main project developed during the internship was **CyberNet**, a cloud-based computer management system deployed on AWS. The system uses services such as Amazon S3, Amazon CloudFront, Amazon VPC, Amazon API Gateway, AWS Lambda, Amazon CloudWatch, AWS IAM, and PayOS integration.

---

## Weekly Worklog Summary

**Week 1:** [Getting familiar with AWS and the First Cloud AI Journey program](1.1-week1/)  
In the first week, I joined the program orientation, learned about the internship requirements, understood the training roadmap, and became familiar with basic AWS Cloud concepts.

**Week 2:** [Learning AWS Cloud fundamentals and basic services](1.2-week2/)  
In this week, I studied cloud computing concepts, AWS global infrastructure, AWS Management Console, IAM, S3, EC2, Lambda, and basic security principles.

**Week 3:** [Preparing AWS account, budget, and development tools](1.3-week3/)  
I prepared the AWS account, configured budget alerts, installed required tools such as VS Code, Git, Node.js, AWS CLI, Hugo, and learned how to manage source code with GitHub.

**Week 4:** [Designing the CyberNet system idea and architecture](1.4-week4/)  
I analyzed the CyberNet project requirements, identified the main system functions, and created the first architecture design for deploying the project on AWS.

**Week 5:** [Creating VPC and basic network infrastructure](1.5-week5/)  
I created the Amazon VPC for CyberNet, configured subnets, route tables, Internet Gateway, and reviewed the resource map of the network infrastructure.

**Week 6:** [Deploying the frontend with Amazon S3 and CloudFront](1.6-week6/)  
I created an S3 bucket, uploaded frontend files, configured static file storage, and prepared CloudFront for frontend content delivery.

**Week 7:** [Building backend logic with AWS Lambda](1.7-week7/)  
I created Lambda functions to handle backend logic for CyberNet, including order processing, payment request handling, and response data for the frontend.

**Week 8:** [Connecting API Gateway with Lambda](1.8-week8/)  
I created API Gateway routes, connected them with Lambda functions, and tested API requests between the frontend and backend.

**Week 9:** [Integrating PayOS payment function](1.9-week9/)  
I studied the PayOS payment flow, configured payment-related environment variables, and prepared the backend logic for creating payment requests.

**Week 10:** [Managing data and resources on Amazon S3](1.10-week10/)  
I organized S3 folders, uploaded product images and static resources, checked object paths, and prepared image URLs for frontend display.

**Week 11:** [Testing CyberNet system components](1.11-week11/)  
I tested the frontend, API Gateway, Lambda functions, PayOS payment flow, S3 resources, and checked CloudWatch Logs to identify possible errors.

**Week 12:** [Finalizing documentation and cleaning up resources](1.12-week12/)  
In the final week, I completed the workshop documentation, updated the internship report, reviewed the project structure, and cleaned up unused AWS resources to control costs.

---

## Result

After 12 weeks of internship, I gained practical experience in AWS Cloud deployment and system documentation.

Through the CyberNet project, I learned how to:

- Design a cloud-based system architecture.
- Deploy frontend files with Amazon S3 and CloudFront.
- Build backend APIs using API Gateway and Lambda.
- Integrate a payment service with PayOS.
- Manage data and static resources on Amazon S3.
- Monitor backend activity with CloudWatch Logs.
- Document the project using Hugo.
- Clean up AWS resources to avoid unnecessary costs.

This worklog reflects the main tasks, progress, and learning outcomes throughout the internship period.
