Phần này tổng hợp nhật ký công việc của tôi trong quá trình tham gia chương trình **First Cloud AI Journey**.

Thời gian thực tập kéo dài 12 tuần, từ ngày **05/05/2026 đến ngày 27/07/2026**. Trong thời gian này, tôi đã tìm hiểu các dịch vụ AWS Cloud, thực hành triển khai hệ thống trên Cloud, thực hiện dự án CyberNet, viết tài liệu Workshop và hoàn thiện báo cáo thực tập.

Dự án chính trong quá trình thực tập là **CyberNet**, hệ thống quản lý phòng máy được triển khai trên AWS. Hệ thống sử dụng các dịch vụ như Amazon S3, Amazon CloudFront, Amazon VPC, Amazon API Gateway, AWS Lambda, Amazon CloudWatch, AWS IAM và tích hợp thanh toán PayOS.

---

## Tổng hợp nhật ký theo tuần

**Tuần 1:** [Làm quen với AWS và chương trình First Cloud AI Journey](1.1-week1/)  
Trong tuần đầu tiên, tôi tham gia buổi định hướng chương trình, tìm hiểu yêu cầu thực tập, nắm bắt lộ trình học tập và làm quen với các khái niệm cơ bản về AWS Cloud.

**Tuần 2:** [Tìm hiểu nền tảng AWS Cloud và các dịch vụ cơ bản](1.2-week2/)  
Tôi tìm hiểu các kiến thức về điện toán đám mây, hạ tầng toàn cầu của AWS, AWS Management Console, IAM, S3, Lambda và các nguyên tắc bảo mật cơ bản.

**Tuần 3:** [Chuẩn bị tài khoản AWS, ngân sách và công cụ phát triển](1.3-week3/)  
Tôi chuẩn bị tài khoản AWS, cấu hình cảnh báo chi phí bằng AWS Budget, cài đặt các công cụ như VS Code, Git, Node.js, AWS CLI, Hugo và làm quen với GitHub.

**Tuần 4:** [Phân tích ý tưởng và thiết kế kiến trúc CyberNet](1.4-week4/)  
Tôi phân tích yêu cầu của dự án CyberNet, xác định các chức năng chính của hệ thống và thiết kế sơ đồ kiến trúc tổng quan để triển khai trên AWS.

**Tuần 5:** [Tạo VPC và hạ tầng mạng cơ bản](1.5-week5/)  
Tôi tạo Amazon VPC cho CyberNet, cấu hình subnet, route table, Internet Gateway và kiểm tra sơ đồ tài nguyên mạng sau khi tạo.

**Tuần 6:** [Triển khai Frontend với Amazon S3 và CloudFront](1.6-week6/)  
Tôi tạo S3 Bucket, upload các file Frontend, cấu hình lưu trữ file tĩnh và chuẩn bị CloudFront để phân phối nội dung website.

**Tuần 7:** [Xây dựng logic Backend với AWS Lambda](1.7-week7/)  
Tôi tạo các Lambda Function để xử lý logic Backend cho CyberNet, bao gồm xử lý đơn hàng, yêu cầu thanh toán và dữ liệu trả về cho Frontend.

**Tuần 8:** [Kết nối API Gateway với Lambda](1.8-week8/)  
Tôi tạo API Gateway, cấu hình route, kết nối API với Lambda Function và kiểm thử request giữa Frontend và Backend.

**Tuần 9:** [Tích hợp chức năng thanh toán PayOS](1.9-week9/)  
Tôi tìm hiểu luồng thanh toán PayOS, cấu hình các biến môi trường cần thiết và chuẩn bị logic Backend để tạo yêu cầu thanh toán.

**Tuần 10:** [Quản lý dữ liệu và tài nguyên trên Amazon S3](1.10-week10/)  
Tôi tổ chức thư mục trên S3, upload hình ảnh sản phẩm và tài nguyên tĩnh, kiểm tra đường dẫn object và chuẩn bị URL hình ảnh để hiển thị trên Frontend.

**Tuần 11:** [Kiểm thử các thành phần của hệ thống CyberNet](1.11-week11/)  
Tôi kiểm thử Frontend, API Gateway, Lambda Function, luồng thanh toán PayOS, tài nguyên trên S3 và kiểm tra CloudWatch Logs để phát hiện lỗi.

**Tuần 12:** [Hoàn thiện tài liệu và dọn dẹp tài nguyên](1.12-week12/)  
Trong tuần cuối, tôi hoàn thiện tài liệu Workshop, cập nhật báo cáo thực tập, kiểm tra lại cấu trúc dự án và dọn dẹp các tài nguyên AWS không còn sử dụng để kiểm soát chi phí.

## Kết quả đạt được

Sau 12 tuần thực tập, tôi đã có thêm kinh nghiệm thực tế trong việc triển khai hệ thống trên AWS Cloud và viết tài liệu kỹ thuật.

Thông qua dự án CyberNet, tôi học được cách:

- Thiết kế kiến trúc hệ thống trên nền tảng Cloud.
- Triển khai Frontend bằng Amazon S3 và CloudFront.
- Xây dựng Backend API bằng API Gateway và Lambda.
- Tích hợp dịch vụ thanh toán PayOS.
- Quản lý dữ liệu và tài nguyên tĩnh trên Amazon S3.
- Theo dõi hoạt động Backend bằng CloudWatch Logs.
- Viết tài liệu dự án bằng Hugo.
- Dọn dẹp tài nguyên AWS để tránh phát sinh chi phí không cần thiết.

Nhật ký công việc này thể hiện quá trình thực hiện, tiến độ và kết quả học tập của tôi trong suốt thời gian thực tập.
