### My Experience

During the First Cloud AI Journey program, I had the opportunity to learn and practice many new concepts related to AWS Cloud. The program helped me better understand how a cloud system is designed, deployed, and managed.

Besides technical knowledge, I also improved my self-learning, teamwork, progress management, and problem-solving skills while working on the project.

### What I Learned

- Gained basic knowledge of AWS Cloud.
- Practiced using Amazon VPC, AWS Lambda, Amazon Cognito, and Amazon S3.
- Learned how to design system architecture diagrams with Draw.io.
- Gained more experience using Git and GitHub.
- Improved my understanding of security, access control, and AWS cost management.

### Challenges

The main challenge was learning many new AWS services within a limited period of time. Some connection and integration issues also required a lot of time to troubleshoot.

Managing the project, workshop documentation, and internship report at the same time occasionally created pressure on the schedule.

### Suggestions

I hope future programs can include more guided hands-on sessions and practical troubleshooting examples. Regular progress reviews would also help teams identify problems and adjust their plans earlier.

Providing several reference architectures or sample projects would make it easier for participants to understand how AWS services are applied in real systems.

### Acknowledgements

I would like to thank the mentors, organizers, and my team members for their guidance and support throughout the internship.

The knowledge and experience gained from this program will be a useful foundation for my continued development in AWS Cloud and cybersecurity.
