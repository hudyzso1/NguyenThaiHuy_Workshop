During my internship in the **First Cloud AI Journey** program from **May 5, 2026 to July 27, 2026**, I had the opportunity to learn about AWS Cloud and apply the knowledge gained to a practical project.

My project focused on developing a cloud-based computer management system. I participated in designing the system architecture, building the workstation application, configuring AWS services, testing system connections, fixing errors, and preparing project documentation.

During the internship, I worked with several technologies and services, including:

- Amazon VPC and Subnets
- AWS Lambda
- Amazon Cognito
- Amazon S3
- Cloud databases
- ElectronJS
- Ngrok
- Git and GitHub
- Draw.io

The internship helped me improve my technical knowledge, teamwork, communication, time management, and problem-solving skills. However, I also recognized several areas that still require further improvement.

### Strengths

During the internship, I always tried to maintain a positive attitude and respect the mentors and other team members.

I was willing to receive feedback and make changes when my work had problems. I also cooperated with the team, supported assigned tasks, and participated in the development and documentation of the project.

My main strengths include:

- Being willing to learn from feedback.
- Maintaining a respectful and professional attitude.
- Cooperating well with team members.
- Trying to complete assigned tasks responsibly.
- Participating in project development and documentation.

### Areas for Improvement

My AWS knowledge and practical experience are still limited, especially when working with multiple services in the same system. I sometimes needed additional time to understand technical documents and resolve integration errors.

I also need to improve my discipline and manage my schedule more carefully. In some stages, balancing project development, Workshop documentation, and the internship report created pressure on my progress.

The areas I need to improve include:

- Strengthening my AWS Cloud knowledge and practical skills.
- Improving discipline and following the planned schedule more consistently.
- Developing a more systematic troubleshooting process.
- Improving technical communication and presentation skills.
- Reading English technical documentation more effectively.
- Managing time better when handling several tasks at the same time.
- Gaining more experience in cloud security and system monitoring.

### Overall Reflection

Overall, the internship was a valuable opportunity for me to experience a practical AWS Cloud project and better understand the process of developing a cloud-based system.

Although my technical skills still need improvement, I gained useful experience in AWS services, system architecture, application integration, teamwork, and project documentation.

The program also helped me recognize my current strengths and weaknesses. In the future, I will continue improving my AWS Cloud knowledge, discipline, communication skills, and problem-solving ability to prepare for work in the fields of cloud computing and cybersecurity.
